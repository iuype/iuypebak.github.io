个人想法关于ssvep要求刺激之间要正交及相位对可分性的贡献的分析

为啥正交的刺激是好的选择？为啥相位对提高分类有影响？思考….

没有信息量的刺激不是好刺激，那么就可以从信息量的角度思考。

核心问题如下：

形式化问题

出发点：希望刺激之间携带更多的信息。信息量与方差成正比。我们可以从方差入手！！

想要分析方差，就需要有方差的表达式。开始推导。
从最简单的情况入手：假设现在有两段经过segment的刺激

方差可以这样估计：

方差$(a(t),b(t)) \
= \sum_{i=1}^{d} (a(i)-c(i))^2 + \sum_{i=1}^{d} (b(i)-c(i))^2 \
= \sum_{i=1}^{d} a(i)^2 + \sum_{i=1}^{d} b(i)^2 +2\sum_{i=1}^{d} c(i)^2 - 2\sum_{i=1}^{d}c(i)(a(i)+ b(i)) \
= \sum_{i=1}^{d} a(i)^2 + \sum_{i=1}^{d} b(i)^2 +2\sum_{i=1}^{d} c(i)^2 - 4\sum_{i=1}^{d}c(i)^2 \\
= \sum_{i=1}^{d} a(i)^2 + \sum_{i=1}^{d} b(i)^2 -2\sum_{i=1}^{d} c(i)^2 \\
= \sum_{i=1}^{d} a(i)^2 + \sum_{i=1}^{d} b(i)^2 -2\sum_{i=1}^{d} (\frac{a(t)+b(t)}{2})^2 \\
= \frac{1}{2}\sum_{i=1}^{d} a(i)^2 + \frac{1}{2}\sum_{i=1}^{d} b(i)^2 - \sum_{i=1}^{d}a(i)b(i) \
= \frac{1}{2}\sum_{i=1}^{d} a(i)^2 + \frac{1}{2}\sum_{i=1}^{d} b(i)^2 -<\mathbb{a},\mathbb{b}>$

所以，
形式化完毕。

方差分析-正交性对方差的贡献

由此可见方差$(a(t),b(t))$由三部分组成:

  1. 信号$a$在时间d内的能量
  2. 信号$b$在时间d内的能量
  3. 信号$a,b$的向量内积

由3.可以看出：为了提高方差，可以人工选取正交的刺激。这样可以使向量内积为0，以期提高方差。

方差分析-相位对方差的贡献

再进一步分析方差：

$\sum_{i=1}^{d} a(i)^2 = \lfloor\frac{d}{l}\rfloor\sum_{i=1}^{l} a(i)^2 + \sum_{i=1}^{d \pmod{l}} a(i)^2 ， l为信号a的周期。$

所以在时间长度d内，信号$a$对方差的贡献可以分成两部分：

  1. 一个周期的能量*在d时间内的包含的完整周期个数
  2. 不足一个周期的残余能量。这部分是由信号的相位决定的。选择一个好的相位可以让这部分贡献增大

当d很大的时候，2.部分的贡献可以忽略不计。
当d很小的时候，2.部分的贡献应该予以重视。

在脑机接口领域中希望提高ITR，是希望d越小越好的，所以引入相位编码可以提升ITR。引入相位是合理的。

综上所述，正交的刺激，经过设计的相位，对提高ITR有好处。
